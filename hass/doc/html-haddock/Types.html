Description

Types module contains all self-defined data types and instances, default representations for records, some type synonyms and lenses. This makes this module necessary to be imported by almost all other hass modules.

data TraverseReqs Source

Record used for efficient passing of arguments to traverseStore function. If onCrypt is specified, traverseStore will skip the subfolders which have their own checkfiles.
